# Employee Workforce Analysis and Visualization

## Project Overview

This project focuses on analyzing employee workforce data to uncover valuable insights related to workforce demographics, employee performance, compensation, engagement, recruitment effectiveness, and retention patterns.

## Objectives

- Analyze workforce demographics and organizational structure.
- Understand employee distribution across departments.
- Evaluate salary and compensation patterns.
- Assess employee performance and satisfaction levels.
- Explore recruitment source effectiveness.
- Identify factors influencing employee engagement and retention.

## Dataset Information

- Total Records: 311 Employees
- Total Features: 35 Columns

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

## Data Preparation

- Checked missing values
- Removed duplicate records

## Key Insights

- Workforce distribution varies across departments.
- Gender and age diversity are present across the organization.
- Salary levels differ based on department and role.
- Employee satisfaction is influenced by factors beyond compensation.
- Higher engagement is generally associated with lower absenteeism.
- Active employees significantly outnumber terminated employees.
- Recruitment effectiveness varies across hiring channels.
- Special project participation supports employee engagement and development.

## Future Recommendations

- Strengthen employee engagement initiatives.
- Improve retention strategies.
- Optimize recruitment channels.
- Encourage participation in special projects.
- Review compensation structures regularly.
- Implement predictive analytics for attrition forecasting.

## Conclusion

The project successfully transformed employee workforce data into meaningful insights that support workforce planning, talent management, employee engagement, and retention strategies.
